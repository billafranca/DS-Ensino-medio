Sobre Nós

O E.E Guiomar Rocha Rinaldi foi fundado em 2000. O objetivo principal da escola é proporcionar uma escola de qualidade para os alunos. A escola possui uma equipe de professores altamente qualificados e experientes, que trabalham em parceria com os alunos para oferecer um ambiente de aprendizagem divertido e motivador. Desde 2020, a escola se tornou período integral (PEI), com o intuito de formar jovens para o mercado de trabalho.

Por que você deve se matricular?

A escola Guiomar Rocha Rinaldi oferece um ambiente de aprendizagem diversificado e motivador, com uma equipe de professores altamente qualificados e experientes. a escola também possui eletivas para os estudantes dos 1° anos, onde os possibilitam de aprender coisas novas e interessantes para os jovens que ingressam no ensino médio. também temos o clube juvenil, onde os jovens aprendem com si mesmos todo tipo de atividade, seja elas de esporte, cultura, aprendizados, etc...

Itinerário Informativo

Desenvolvimento de Sistemas

Desenvolvimento de Sistemas é um curso extremamente abrangente, onde estudamos e aplicamos todos os conceitos de programação, desde o lógica de programação, algoritmos, fundamentos de programação, etc... o curso também proporciona o conhecimento em banco de dados, programação orientada a objetos, sistemas operacionais, redes de computadores, entre outros. ou seja, aqui trabalhamos com desenvolvimento de software e sistemas embarcados, tudo isso com o intuito de formar profissionais qualificados e competentes para o mercado de trabalho.
